# This is a Expense Report App using nextjs and material ui

## Getting Started

Using local node server, run the development server:

1. install dependencies

```bash
npm install
# or
yarn
```

1. start local dev server

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the expense app live.

Using docker-compose setup, just do:

```bash
docker-compose up
```
